## Getting Started

1. **Clone the repository**
```bash
git clone https://github.com/your-username/your-social-links-site.git
````

2. **Customize your content**

* Update bio, welcome message, and social links
* Replace background video or playlist

3. **Setup Discord PFP Sync (Optional)**

* Deploy the Cloudflare Worker from [Discord PFP Fetcher](https://github.com/e-lyas/Discord-PFP-Fetcher)
* Replace `userId` and add your Discord Bot Token

4. **Deploy**

* Host on **Vercel**, **Netlify**, or **GitHub Pages**
